# Retail-Sales-Discount-Impact-Analysis-Power-BI
 Developed interactive Power BI dashboard to analyze sales performance, product demand, and discount
impact for a multi-store retail company (300-record dataset).
 - Cleaned and transformed data using Python (Pandas); implemented DAX measures for key metrics.
 - Designed dashboards with KPI cards, charts, and maps to drive data-backed decisions.
 - Key insights: sales trends, top performers, discount impact, and seasonality.
 - Identified opportunities to optimize discount strategies and improve revenue growth.
 - Presented findings to stakeholders, enabling data-driven business decisions.
